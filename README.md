# 3D 粒子动画工作台

一个纯前端的浏览器工具，用于预览 3D 模型并把模型几何生成实时粒子动画。

## 使用方式

推荐启动本地静态服务器：

```bash
node server.js
```

然后打开：

```text
http://localhost:5173
```

也可以直接双击 `index.html` 打开，但部分浏览器对模块和模型资源会有更严格限制。

## 支持格式

- `.glb`：推荐，最稳定
- `.gltf`：如果有 `.bin` 或贴图文件，请一起多选
- `.obj`：支持基础几何预览

## 功能

- 浏览器内 3D 模型预览
- 拖拽或选择本地模型文件
- 从模型顶点采样生成粒子
- 默认参考蓝色线框车身向橙色粒子羽流解体的视觉风格
- 溶解、爆裂、环绕三种动画模式
- 粒子数量、速度、扩散、尺寸、双色通道控制
- 风力模拟：风力强度、方向、高度、湍流强度、随机性
- 粒子物理：吸引力、排斥力、摩擦力
- 模型显示模式：材质、线框、混合
- 自定义材质颜色、金属度、粗糙度、透明度与线框颜色
- 自定义主光、环境光、补光、轮廓光、光源方位、高度与曝光
- 保存当前参数为 JSON 预设
