# 3D 粒子动画工作台

一个纯前端的浏览器工具：把 3D 模型当作"发射源"，从模型表面持续生成可控的粒子流，配合风、湍流、漩涡、群聚等力场组合做出像羽流、星尘、丝缕那样的效果。可录像、可导出粒子点云到 DCC 软件继续处理。

---

## 一分钟跑起来

```bash
node server.js
```

浏览器打开 `http://localhost:5173`，会先看到一个示例模型（demo shape），所有控件都已可用，可以一边拖参数一边看效果。

> Windows 下也可以双击仓库根目录的 `启动.bat`，会自动起服务并打开浏览器。

## 右侧面板详解

### 模型

- **选择模型文件**：点上传框选本地文件。`.gltf` 拆分文件请把 `.gltf + .bin + 贴图` **一次性多选**全部，否则贴图/几何会丢
- **顶点 / 粒子**：实时显示当前模型顶点数和正在跑的粒子数

### 显示

控制原始模型怎么呈现（**与粒子无关**，纯视觉）：

| 控件 | 说明 |
|---|---|
| **材质 / 线框 / 混合** | 三选一。混合 = 材质 + 半透明线框叠加，一般最好看 |
| **材质色 / 线框色** | 颜色独立可调 |
| **金属度 / 粗糙度** | 标准 PBR 参数 |
| **模型透明度 / 线框透明度** | 模型默认偏透（0.2），让粒子穿透感更强 |

### 粒子

这是**核心面板**。

**采样模式**（粒子从模型的哪里生出来）：

- **表面**（默认）：按三角面面积加权随机采样，分布最均匀，最像"表面发射"
- **顶点**：只在模型原始顶点上生粒子。模型尖角/转折处会聚集
- **边**：沿三角形的边随机采样，做出**线框流光**效果
- 切换会**立刻重建**所有粒子，不需要等

| 参数 | 含义 |
|---|---|
| **粒子数量** | 4 千 ~ 400 万。10 万以下绝大多数显卡都流畅；100 万 + 建议关闭群聚 modifier |
| **时间倍率** | 整个模拟的播放速度，0.1 慢动作，3 倍速狂野 |
| **粒子尺寸 / 尺寸抖动** | 抖动 > 0 时每个粒子尺寸略不同，更自然 |
| **寿命 / 寿命抖动** | 粒子从生到死的秒数。**改了只对新生粒子生效**——已经活着的粒子保留出生时的寿命，所以参数会用一个寿命周期慢慢"渗透"完成 |
| **表面初速** | 粒子刚出生时沿表面切向飞出去的速度。这是为什么粒子看起来"先贴着模型滑一下再被风吹走" |

### Modifier 栈（力场组合）

像 X-Particles / Cinema 4D 的修改器栈——添加多个力，**叠加生效**。

下拉选类型 → 点 **+ 添加** → 出现一行可启用/禁用/删除/调参：

| 类型 | 作用 | 典型用法 |
|---|---|---|
| **风** | 全局方向力（强度 / 方位角 / 仰角） | 主流向，几乎每个场景都加 |
| **湍流** | 3D 噪声，模拟乱流（强度 / 尺度 / 倍频） | 让粒子轨迹"乱"起来不机械 |
| **阻力** | 速度衰减，让粒子减速 | 加在最后稳住整体节奏 |
| **重力** | Y 轴加速度（负值变上升） | 烟雾、火焰、坠落感 |
| **漩涡** | 绕指定轴旋转（强度 / 轴方位 / 轴仰角） | 做龙卷、轨道、星云 |
| **随机扰动** | 每帧无相关随机推力 | 静态颗粒感（注意：会让相邻粒子失去相关性，不要堆太大） |
| **表面捕获** | 把粒子拉回出生点 + 衰减法向速度 | 让粒子贴着模型表面"流动"而不飞走 |
| **群聚** | 邻域聚拢 + 速度对齐 + 近距避让 | 做丝缕、鱼群、银河条带（**CPU 密集，粒子数 > 50 万会卡**） |

> 没有任何力场时，粒子只受出生速度推动 → 一片缓缓飘散。

### 颜色（Ramp）

**颜色模式**：

- **按年龄**：粒子从生到死按渐变色变化（最常用，做火苗/羽流）
- **按速度**：速度越大颜色越靠 ramp 末端（突显高速粒子）
- **固定**：所有粒子用同一段 ramp（看起来一致）

**色标编辑**：

- 渐变条下方点 **+ 色标** 添加新点，**双击色标小方块**改颜色，**拖动**改位置，右键删除
- 至少要保留 2 个色标
- **速度参考(最大)**：仅在"按速度"模式下生效，定义"满 ramp"对应的速度阈值
- **亮度衰减**：值越大，老粒子越暗（视觉上的淡出）。和加色混合配合就是免费的"溶解"效果
- **显示原始模型 / 显示地面网格**：纯显示开关

### 光影

实时影响**模型**的光照（粒子是 additive blending，不被这些光影响）：

- **主光 / 环境 / 补光 / 轮廓光**：四个独立强度
- **主光方位 / 高度**：方位 = 水平角度，高度 = 仰角
- **曝光**：整体亮度，相当于相机曝光补偿

### 预设

- 起个英文名（比如 `storm-on-statue`）→ **保存当前**
- 保存的预设会出现在下方列表，点击**应用**即可一键还原
- 预设存在**浏览器 localStorage**——换浏览器/清缓存会丢，**正式备份请用 `导出预设` 工具栏按钮（生成 JSON 文件）**

> **保存 ≠ 导出**：保存只在本机本浏览器；导出是写出 `.json` 文件可以拷贝到别的机器上再导入。

## 时间轴

- **暂停/播放**：冻结/恢复粒子模拟（模型旋转视角不受影响，可以暂停后慢慢调相机）
- **进度滑条**：当前未实现 scrub 回放（粒子是实时模拟非动画曲线），主要供录制时定位
- **FPS**：右下角实时帧率，掉到 30 以下就该减粒子数或关 cohesion

## 拖拽加载

直接把文件**拖进 3D 视口**：

- 单文件：`.glb` / `.obj` 直接读
- 多文件：`.gltf` 必须连同 `.bin` + 贴图一起拖（拖一个文件夹也行，浏览器会展开）

| 格式 | 说明 |
|---|---|
| `.glb` | **推荐**，所有数据打包在一个文件 |
| `.gltf` | 必须连同 `.bin` 和贴图一起选 |

## 几个常见配方

**像车身解体的羽流**
- 采样：表面 / 寿命 2.2 / 表面初速 1.2
- 力场：风（仰角 +8）+ 湍流（强度 0.9 / 尺度 1.5）+ 阻力 1.4
- 颜色：按年龄，从冷蓝 → 橙黄 → 透明

**贴着模型流动的丝绸**
- 采样：表面 / 寿命 3.0
- 力场：表面捕获（强度 3 / 法向阻尼 0.85）+ 湍流（弱，0.4）
- 颜色：按速度，慢的暗、快的亮

**线框流光**
- 采样：边 / 数量 8 万 / 表面初速 0.3
- 力场：风（弱）+ 阻力（高）
- 颜色：固定，单色高亮

**龙卷风**
- 采样：表面
- 力场：漩涡（轴仰角 90，绕 Y 轴）+ 重力（负值，上升）+ 湍流
- 颜色：按速度

## 性能参考

| 粒子数 | 中端独显 | 集成显卡 |
|---|---|---|
| 5 万以下 | 60+ fps | 60 fps |
| 20 万 | 60 fps | 30~50 fps |
| 100 万 | 50 fps（不开 cohesion） | 卡顿 |
| 400 万上限 | 视显卡而定，建议关闭 cohesion + 阻力以外的复杂力场 |

群聚 modifier 是 O(N²) 邻域查询的简化版，跑大粒子数时**单独关掉它**就能恢复流畅。

## 导出到 DCC 软件

点 **导出 PLY** 后会下载一个 `.ply` 文件，含 xyz + RGB。

- **C4D**：Volume Builder 输入 → Volume Mesher → Octane / Redshift 体积着色
- **Houdini**：File SOP 直接读 → vdbfromparticles → VDB 渲染
- **Blender**：Geometry Nodes → Points to Volume

> 浏览器直接生成 VDB 不实用，PLY → DCC 是稳定的转换路径。

## 故障排查

| 现象 | 原因 / 解决 |
|---|---|
| 拖入 `.gltf` 后贴图全黑 | 没有同时多选 `.bin` 和贴图。重新点上传框 ctrl+多选所有相关文件 |
| 粒子一加载就飞出屏幕 | 模型超大或偏离原点。代码内部 `normalizeModel` 会自动归一化，如果你绕过了它就会出现这种情况 |
| FPS 暴跌 | 第一步：关掉**群聚** modifier；第二步：降粒子数；第三步：关闭线框（混合模式有两套材质） |
| 录制视频文件很大 | 默认 16 Mbps、`.webm`，可在 `app.js` 里改 `videoBitsPerSecond` |
| 浏览器报 ES module 导入错误 | 一定要用 `node server.js` 启动，**不要直接双击 `index.html`**——浏览器对 file:// 下的 ES 模块限制很严 |

## 技术栈

- Three.js 0.164.1（通过 importmap 从 jsDelivr CDN 加载，**无构建步骤、无 npm 依赖**）
- 单 ES module（`app.js`）+ 可选 GPU 模拟（`gpuSim.js`）
- Node 内置 http 模块的极简静态服务器（`server.js`）

详细架构说明见仓库内 `CLAUDE.md`。
